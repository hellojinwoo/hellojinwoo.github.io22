    Q1. Is there a relationship between a predictor and a dependant variable?¶

    $$ \text{Life Expectancy} = \hat{\beta}_0 + \hat{\beta}_1 * \text{Spending USD} + \epsilon $$
    • To validate if there is a relationship between life_exptectancy and Spending_USD, we can check if $\hat{\beta}_1$ is 0 or not.
      • Let's suppose $\hat{\beta}_1 = 0$, and if the p-value of this null hypothesis is below the significance level, 0.05 (5%), which is the value that we usually choose.
